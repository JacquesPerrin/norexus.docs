# NOREXUS Docs

Plain Fumadocs documentation site for NOREXUS CLI, Auth, and ID.

## Development

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000/docs`.

## Build

```bash
pnpm build
```

## Content

Docs live in `content/docs`.

Add new pages to `content/docs/meta.json` so they appear in the sidebar.
