NOREXUS Docs is the documentation home for the terminal-first CLI, Auth, and ID surfaces.

## Products

| Product | Domain | Owns |
| --- | --- | --- |
| CLI | `cli.norexus.app` | Terminal workflows, agent routing, MCP presets, local diagnostics, and developer commands. |
| Auth | `auth.norexus.app` | Login, CLI pairing, account approval, session exchange, and identity consent flows. |
| ID | `id.norexus.app` | Signed-in identity settings, organizations, security state, service access, and account identity. |

## Start here

- Use [CLI](/docs/cli) to install `nrx` and understand the command flow.
- Use [Auth](/docs/auth) to understand login and CLI pairing.
- Use [ID](/docs/id) to understand identity settings and organization state.
- Use [Agent context](/docs/agent-context) when handing this docs repo to another coding agent.
